## Скрипт auto-deploy.sh выполняет следующее интерактивные действия:
- Предварительное создание директории для будущего размещения проекта
- Добавление адреса ресурса для последующего клонирования репозитория
- Формирование env файла с переменным окружением, для старта mysql, а также использование их для подключения к DB из приложения.
- Автоматический deploy compose.yaml проекта.
